# reviews-affinity-clustering-llm

End-to-end пайплайн: сбор публичных отзывов с геосервиса, affinity-анализ пересечения аудиторий брендов, кластеризация предложений и интерпретация тем с помощью LLM (Ollama) для получения практических инсайтов.

## Структура проекта

- `notebooks/step1_parser_yk.ipynb` — **Шаг 1. Парсинг**: точки бренда → отзывы → авторы → отзывы авторов по другим местам  
- `notebooks/step2_affinity_analysis.ipynb` — **Шаг 2. Affinity-анализ**: пересечение аудитории брендов с брендом-фокусом 
- `notebooks/step3_clustering_llm.ipynb` — **Шаг 3. Кластеризация предложений + LLM-интерпретация**: эмбеддинги, KMeans, описания тем и профилей брендов + интерактивный просмотр
