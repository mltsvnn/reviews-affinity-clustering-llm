# reviews-affinity-clustering-llm

End-to-end пайплайн: сбор отзывов с Яндекс Карт → affinity-анализ пересечения аудиторий брендов → кластеризация предложений и интерпретация тем с помощью LLM (Ollama) для получения практических инсайтов.

## Структура проекта

- `блокноты/step1_parser_yk.ipynb` — **Шаг 1. Парсинг**: точки бренда → отзывы → авторы → отзывы авторов по другим местам  
- `блокноты/step2_affinity_analysis.ipynb` — **Шаг 2. Affinity-анализ**: пересечение аудитории брендов с брендом-фокусом («Золотое Яблоко») + опциональная поправка на внешний трафик  
- `блокноты/step3_clustering_llm.ipynb` — **Шаг 3. Кластеризация и LLM-интерпретация тем**: эмбеддинги → KMeans → описания тем + профили брендов по темам + интерактивный просмотр
